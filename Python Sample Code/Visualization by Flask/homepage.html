Analyzing and Predicting
NYC Citi Bike

About

As an affordable and sustainable mode of travel, the bike share system has been demonstrated to reduce traffic congestion, decrease air pollution, enhance physical activity, and complement public transport. Citi Bike riders in NYC broke a monthly ridership record of over 4 million trips in August 2023. This project aims to analyze Citi Bike usage in NYC, focusing on creating an interactive dashboard to visualize spatial-temporal patterns of bike trips and applying machine learning models to predict daily Citi Bike ridership based on weather and temporal features.

Our project aims to improve operational efficiency and urban planning, leading to enhanced system performance and user satisfaction. Operators of Citi Bike and similar systems can benefit from these tools to better understand usage patterns, predict peak demand periods, and ensure bikes are available where and when they are needed. Additionally, the project aims to enhance the daily experience of users who rely on bikes for commuting, contributing to healthier lifestyles and reducing environmental impact.

Visualization

• Visualize Bike Share Usage

We have developed an interactive dashboard to visualize the spatial and temporal patterns of Citi Bike trips. This dashboard includes a variety of visualization tools to explore data in different formats, such as:

  • Line charts
  • Bar charts and stacked bar charts
  • Maps with customizable filters for detailed data analysis

Plot 1: Monthly Aggregation Bar Chart

This interactive bar chart aggregates Citi Bike trip data by month. It includes multiple options for users to filter and customize the data view by clicking the âš™ button. The interactive design, including hover values and customizable settings, enables detailed exploration of monthly ride data, providing insights into variations and usage trends across different bike types, metrics, and membership categories.

  • Chart Type: Displays bar charts, with optional stacked bars for a deeper breakdown.
  • Moving Average Line: A six-month moving average line is included to help highlight trends. It can be toggled on or off.

Plot 2: Station-Level Trip Counts

This interactive plot shows Citi Bike station trip counts across New York City. It features a map of stations represented by circles, as well as two bar charts displaying the top 10 pickup and return stations for the selected year.This interactive plot helps users explore detailed station-level trip data across different timeframes, making it easier to identify high-usage stations and usage patterns throughout New York City.

  • Map Display:
    • Circle Markers: Each station is represented by a circle. Size and color intensity indicate trip counts, with darker, larger circles representing higher volumes.

  • Year Filter: Users can filter the bar chart data by year, enabling them to observe trends and changes over time.

Plot 3: Station Connectivity Map

This interactive map provides a dynamic visualization of Citi Bike stations, featuring hover details and connection insights. This setup allows users to explore Citi Bike station connectivity and travel patterns, offering both individual station information and network-level insights when hovering over different stations.

Prediction

• Predict Bike Share Ridership

Predicting bike share ridership is crucial for effective transportation demand management. We applied machine learning models in Python to predict daily Citi Bike ridership, considering various weather and temporal features. These predictions help in planning resources to ensure availability and improve operational efficiency.

Plot 4.1: Averaged Trip Duration Prediction with 14-Day Moving Average

This plot illustrates the actual versus predicted Citi Bike trip durations over time using a 14-day moving average to smooth out the data trends. The chart includes both the training and testing periods, clearly divided for clarity.

  • Axes:
    • X-Axis (Date): Displays the timeline from early 2022 to late 2024.
    • Y-Axis (Averaged Trip Duration in Minutes): Represents the average trip duration, ranging from 8 to 19 minutes.
  • Training vs. Testing Period:
    • The chart is divided into training and testing periods, indicated by a vertical dashed line. The left side represents the training set (blue background), while the right side represents the testing set (pink background).
  • 14-Day Moving Average Lines:
    • Actual Trip Duration: Blue line.
    • Linear Regression Predictions: Yellow line.
    • Random Forest Predictions: Green line.
    • K-Nearest Neighbors (KNN) Predictions: Purple line.
    • Gradient Boosting Predictions: Red line.
  • Observations:
    • The 14-day moving averages provide a clearer view of trends by reducing noise in the data.
    • Most prediction models align closely with actual trends, though deviations can be observed, particularly with KNN and Gradient Boosting models at certain periods.
    • The legend displays Adjusted R-squared values for each model. The Linear Regression model shows the highest Adjusted R-squared value (0.80), indicating it best captures the underlying linear trends in the data.

Plot 4.2: Trip Count Prediction with 14-Day Moving Average

This plot presents the actual versus predicted daily trip counts using a 14-day moving average to visualize overall patterns more clearly. The chart is divided into training and testing segments.

  • Axes:
    • X-Axis (Date): Displays the timeline from early 2022 to late 2024.
    • Y-Axis (Trip Count): Represents the daily trip count, ranging from approximately 20,000 to 200,000 trips.
  • Training vs. Testing Period:
    • The chart is divided into training and testing periods, indicated by a vertical dashed line. The left side represents the training set (blue background), while the right side represents the testing set (pink background).
  • 14-Day Moving Average Lines:
    • Actual Trip Count: Blue line showing the actual recorded daily trip counts over time.
    • Linear Regression Predictions: Orange line.
    • Random Forest Predictions: Green line.
    • K-Nearest Neighbors (KNN) Predictions: Purple line.
    • Gradient Boosting Predictions: Red line.
  • Observations:
    • The 14-day moving averages effectively reduce data noise, helping to highlight general trends.
    • Random Forest and Gradient Boosting models align well with actual data trends, while the KNN and Linear Regression models show slight deviations during certain periods.
    • The legend shows Adjusted R-squared values for each model. The Gradient Boosting model has the highest Adjusted R-squared value (0.88), suggesting its strong ability to capture complex, non-linear relationships in the data.
